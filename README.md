# CreditWise Loan Approval System (Machine Learning)

## Problem Statement
Manual loan approval processes are slow, biased, and inconsistent, leading to
financial losses and rejection of genuine customers.

## Solution
An intelligent ML-based system that predicts loan approval using applicant
financial and demographic data.

## Techniques Used
- Data Cleaning & Missing Value Handling
- Exploratory Data Analysis (EDA)
- One-Hot Encoding
- Correlation Analysis
- Logistic Regression
- KNN Classifier
- Naive Bayes Classifier

## Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Lab

## Outcome
The system provides accurate, unbiased, and fast loan approval predictions.
